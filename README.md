## 🇬🇧 English

### About
NoteApp is an open source note-taking app for Android, inspired by Google Keep. It was created as a personal project to replace Google Keep with a lightweight.
Built with **Kotlin** and **Jetpack Compose**.

### Features
- 📝 Plain text notes
- ✅ Checklist / To-do lists
- 🖼️ Attach images to notes
- 🌙 Dark / Light mode
- 🌐 Spanish and English support
- 📤 Share notes via WhatsApp, email, Bluetooth and more
- 💾 Local storage with Room database
- ⚙️ User preferences with DataStore

### 📋 Requirements
- Android 13 (API 33) or higher
- ~10 MB of free storage

### 📥 Installation

#### Option 1 — Download APK (recommended)
1. Go to the [Releases](https://github.com/thejv04/NoteApp/releases) page
2. Download the latest `app-debug.apk`
3. On your Android device go to **Settings → Security → Install unknown apps** and allow your browser or file manager
4. Open the downloaded APK and tap **Install**

#### Option 2 — Build from source
1. Clone the repository:
```bash
git clone https://github.com/thejv04/NoteApp.git
```
2. Open the project in **Android Studio**
3. Wait for Gradle to sync
4. Connect your device or start an emulator
5. Click **Run** ▶️

### Acerca de
NoteApp es una app de notas de código abierto para Android, inspirada en Google Keep. Nació como un proyecto personal para reemplazar Google Keep con una alternativa ligera.
Construida con **Kotlin** y **Jetpack Compose**.

### Funcionalidades
- 📝 Notas de texto libre
- ✅ Listas de tareas con checklist
- 🖼️ Adjuntar imágenes a las notas
- 🌙 Modo oscuro / claro
- 🌐 Soporte para español e inglés
- 📤 Compartir notas por WhatsApp, correo, Bluetooth y más
- 💾 Almacenamiento local con base de datos Room
- ⚙️ Preferencias de usuario con DataStore

### 📋 Requisitos
- Android 13 (API 33) o superior
- ~10 MB de almacenamiento libre

### 📥 Instalación

#### Opción 1 — Descargar APK (recomendado)
1. Ve a la página de [Releases](https://github.com/thejv04/NoteApp/releases)
2. Descarga el último `app-debug.apk`
3. En tu dispositivo Android ve a **Ajustes → Seguridad → Instalar apps desconocidas** y permite tu navegador o gestor de archivos
4. Abre el APK descargado y toca **Instalar**

#### Opción 2 — Compilar desde el código fuente
1. Clona el repositorio:
```bash
git clone https://github.com/thejv04/NoteApp.git
```
2. Abre el proyecto en **Android Studio**
3. Espera a que Gradle sincronice
4. Conecta tu dispositivo o inicia un emulador
5. Haz clic en **Ejecutar** ▶️
